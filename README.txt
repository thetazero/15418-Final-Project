15-418 Final Project: Connect 5 Engine

Executables To Run:

./run_engine <board_file>.txt
- This will load a board file and run the engine to get recommended lines on that position.

./engine_vs_random <depth>
- This will have the engine play against random moves searching at the given depth.
- The default depth is 5.

./engine_vs_engine <depth_x> <depth_o>
- This will have two engines play against each other searching at the given depths.
- The default depths are 5 for both engine.
